Pirate Cave

Pirate Cave é um Jogo 2D de plataforma, onde seu objetivo é encontrar a Caveira Sagrada, após derrotar o poderoso Shark.

Sobre O Jogo

Engine: Construct 3

Arte: Pixel Arte, 2D, By Pixel Frog

Gênero: Plataforma

Sistema: Browser, Windows

Desenvolvedor: Marcos Tabachini
